Follow these steps to create a library:

  1. Create a new project.
  2. In the project panel, click Add Library.
  3. Click Create New.
  4. In the Input Library Name field, enter a name for the library (for example, Calculators).

    5. To create a library directory, name the name to directoryName / libraryName (for example, Tools / Calculators).

  5. Click Create Library.

    6. The template library files are added to your project. View the files in the Explorer panel.

  6. In the right navigation menu, click the Explorer icon.
  7. In Explorer panel, open the Library.pyLibrary.cs file, rename it to reflect its purpose (e.g.: TaxesCalculator.pyTaxesCalculator.cs), and implement your library.
