You must set up the algorithm to check the portfolio state before you can place buy and hold orders.

To place the orders, in the body of the condition that checks if the portfolio is not invested, check if the Slice contains data for each Equity. If the Slice contains data for an Equity, call the SetHoldings method with the Equity Symbol and a portfolio weight.

foreach (var symbol in new [] {_spy, _msft, _googl})
{
    if (data.Bars.ContainsKey(symbol))
    {
        SetHoldings(symbol, 0.33);
    }
}
for symbol in [self.spy, self.msft, self.googl]:
    if symbol in data.Bars:
        self.SetHoldings(symbol, 0.33);
