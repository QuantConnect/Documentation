You can request historical Options data by using the data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no data in the period you request, the history result will be empty. The following example gets the historical data for US Options by using the History API.

history = self.History(contract.Symbol, 10, Resolution.Minute)
var history = History(contract.Symbol, 10, Resolution.Minute);

You can request historical Options data in the Research environment by using the GetOptionHistory method of the QuantBook class. To learn more about historical data requests for Options in the Research environment, please visit the Research Historical Data documentation. The following example gets the historical data for US Options in the Research environment.

qb = QuantBook()
option = qb.AddOption("GOOG") 
option.SetFilter(-2, +2, timedelta(0), timedelta(180))
history = qb.GetOptionHistory(option.Symbol, datetime(2020, 6, 1), datetime(2020, 6, 5))
history.GetAllData()
var qb = new QuantBook();
var option = qb.AddOption("GOOG");
option.SetFilter(-2, +2, TimeSpan.FromDays(0), TimeSpan.FromDays(180));
var history = qb.GetOptionHistory(option.Symbol, new DateTime(2020, 6, 1), new DateTime(2020, 6, 5));
var historicalOptionData = history.GetAllData();
