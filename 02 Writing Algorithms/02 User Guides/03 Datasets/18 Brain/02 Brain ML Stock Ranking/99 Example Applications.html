The Brain ML Stock Ranking dataset enables researchers to test strategies using the machine learning ranking provided by Brain also in combination with other datasets. Examples to be tested include:

  • Constructing a portfolio of securities with each security's weight in the portfolio reflecting its Brain ML Rank
  • Test a strategy that goes long a certain number of stocks at the top of the Brain ML Stock Ranking
  • Build a market-neutral strategy based on the top N and bottom N stocks ranked by the dataset
