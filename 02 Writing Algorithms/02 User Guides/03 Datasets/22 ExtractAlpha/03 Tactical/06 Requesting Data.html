To add Tactical dataset by ExtractAlpha to your algorithm, use the AddData() method to request the data. As with all datasets, you should save a reference to your symbol for easy use later in your algorithm. For detailed documentation on using custom data, see Importing Custom Data.

class ExtractAlphaTacticalModelDataAlgorithm(QCAlgorithm):
    def Initialize(self):
        self.SetStartDate(2019, 1, 1)
        self.SetEndDate(2020, 6, 1)
        self.SetCash(100000)

        aapl = self.AddEquity("AAPL", Resolution.Daily).Symbol
        extract_alpha_tactical_model_symbol = self.AddData(ExtractAlphaTacticalModel, aapl).Symbol
namespace QuantConnect
{
    public class ExtractAlphaTacticalModelDataAlgorithm : QCAlgorithm
