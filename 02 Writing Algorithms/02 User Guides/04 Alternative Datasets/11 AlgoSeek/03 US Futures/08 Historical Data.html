You can request historical Futures data by using the data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no data in the period you request, the history result will be empty. The following example gets the historical data for US Futures by using the History API.

history = self.History(contract.Symbol, 10, Resolution.Minute)
var history = History(contract.Symbol, 10, Resolution.Minute);

You can request historical Futures data in the Research environment by using the GetFutureHistory method of the QuantBook class. To learn more about historical data requests for Futures in the Research environment, please visit the Research Historical Data documentation. The following example gets the historical data for US Futures in the Research environment.

qb = QuantBook()
gold = qb.AddFuture(Futures.Metals.Gold) 
gold.SetFilter(0, 90)
history = qb.GetFutureHistory(gold.Symbol, datetime(2020, 6, 1), datetime(2020, 6, 5))
history.GetAllData()
var qb = new QuantBook();
var gold = qb.AddFuture(Futures.Metals.Gold);
gold.SetFilter(0, 90);
var history = qb.GetFutureHistory(gold.Symbol, new DateTime(2020, 6, 1), new DateTime(2020, 6, 5));
var historicalFutureData = history.GetAllData();
