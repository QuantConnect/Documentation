To add CBOE data to your algorithm, use the AddData() method to request the data. As with all datasets, you should save a reference to your symbol for easy use later in your algorithm. For detailed documentation on using custom data, see Importing Custom Data.

class CboeDataAlgorithm(QCAlgorithm):
 
    def Initialize(self):
        self.SetStartDate(2003, 1, 1)
        self.SetEndDate(2019, 10, 11)
        self.SetCash(100000)
 
        self.cboeVix = self.AddData(CBOE, "VIX", Resolution.Daily).Symbol
