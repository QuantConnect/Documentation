You can request historical custom data in your algorithm using the custom data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no custom data in the period you request, the history result will be empty. The following example gets the historical data for Benzinga News by using the History API.

history = self.History(self.benzinga_symbol, 14, Resolution.Daily)
var history = History<BenzingaNews>(_benzingaSymbol, 14, Resolution.Daily);
