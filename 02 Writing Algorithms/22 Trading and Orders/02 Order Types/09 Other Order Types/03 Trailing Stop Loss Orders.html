A trailing stop loss order is a dynamic stop loss that adjusts towards the security price as the security moves away from the stop price. For example, if you set a trialing stop loss to 10%, the furthest away it will ever be from the asset price is 10%. If it's a sell trailing stop loss order that's below the asset price and the asset price increases such that the trailing stop is more than 10% away, the trailing stop loss moves up to be 10% below the asset price. If the asset prices decreases, the sell trailing stop loss doesn't move.
