Indicators are not ready when you first create them. The length of time it takes to trust the indicator values depends on the indicator period. In QuantConnect, we provide a shortcut to check the indicator status. indicator.IsReady will return true when the indicator is ready to be used. eg. $[IsReady,P:QuantConnect.Indicators.Indicator.IsReady] 

	if not self.indicator.IsReady:
		return

	if (!indicator.IsReady) return;
