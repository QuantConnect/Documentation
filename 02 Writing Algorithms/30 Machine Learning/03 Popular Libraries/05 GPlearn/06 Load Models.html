You can load and trade with pre-trained GPlearn models that saved in ObjectStore. To do so, follow these steps to load them:

  1. Check if the ObjectStore already has the model saved during Initialize with the model key.
    2. def Initialize(self) -> None:
    if self.ObjectStore.ContainsKey(transformer_model_key) and self.ObjectStore.ContainsKey(regressor_model_key):

    This method returns a boolean that represents if the transformer_model_key and regressor_model_key are in the ObjectStore. If the ObjectStore does not contain them, save the models using them before you proceed.

  2. Call the GetFilePath method with the key names.
    3.         transformer_file_name = self.ObjectStore.GetFilePath(transformer_model_key)
            regressor_file_name = self.ObjectStore.GetFilePath(regressor_model_key)

    This method returns the paths where the models are stored.

  3. Call the load method with the file path.
    4.          self.gp_transformer = joblib.load(transformer_model_key)
            self.model = joblib.load(regressor_file_name)

    This method returns the saved models.
