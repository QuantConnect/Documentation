To build the Range bars with a different property than the Value of the IBaseData object, provide a selector argument. The selector should be a function that receives the IBaseData object and returns a decimal value.

self.consolidator = ClassicRangeConsolidator(1, selector = lambda data: data.High)
_consolidator = new ClassicRangeConsolidator(1m, data => (data as TradeBar).High);

To add a non-zero Volume property to the Range bars, provide a volumeSelector argument. The volumeSelector should be a function that receives the IBaseData object and returns a decimal value.

self.consolidator = ClassicRangeConsolidator(1, volumeSelector = lambda data: data.Volume)
_consolidator = new ClassicRangeConsolidator(1m, null, data => (data as TradeBar).Volume);
