The following table describes the MarketDataType enumerator members:

Member Value Description
Base 0 Base market data type
TradeBar 1 TradeBar market data type (OHLC summary bar)
Tick 2 Tick market data type (price-time pair)
Auxiliary 3 Data associated with an instrument
QuoteBar 4 QuoteBar market data type [Bid(OHLC), Ask(OHLC) and Mid(OHLC) summary bar]
OptionChain 5 Option chain data
FuturesChain 6 Futures chain data
