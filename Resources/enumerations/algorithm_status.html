The following table describes the AlgorithmStatus enumeration members:

Member Value Description
DeployError 0 Error compiling algorithm at start (0)
InQueue 1 Waiting for a server (1)
Running 2 Running algorithm (2)
Stopped 3 Stopped algorithm or exited with runtime errors (3)
Liquidated 4 Liquidated algorithm (4)
Deleted 5 Algorithm has been deleted (5)
Completed 6 Algorithm completed running (6)
RuntimeError 7 Runtime Error Stoped Algorithm (7)
Invalid 8 Error in the algorithm id (not used) (8)
LoggingIn 9 The algorithm is logging into the brokerage (9)
Initializing 10 The algorithm is initializing (10)
History 11 History status update (11)
