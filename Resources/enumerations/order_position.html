The following table describes the OrderPosition enumeration members:

Member Value Description
BuyToOpen 0 Indicates the buy order will result in a long position, starting either from zero or an existing long position (0)
BuyToClose 1 Indicates the buy order is starting from an existing short position, resulting in a closed or long position (1)
SellToOpen 2 Indicates the sell order will result in a short position, starting either from zero or an existing short position (2)
SellToClose 3 Indicates the sell order is starting from an existing long position, resulting in a closed or short position (3)
