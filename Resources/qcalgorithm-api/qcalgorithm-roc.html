roc(symbol, period, resolution=None, selector=None)[source]

Creates a new RateOfChange indicator. This will compute the n-period rate of change in the security. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The symbol whose PPHL we seek
  • period (int) — The number of surrounding bars whose high values should be less than the current bar's for the bar high to be marked as high pivot point
  • resolution (Resolution, optional) — The number of surrounding bars whose low values should be more than the current bar's for the bar low to be marked as low pivot point
  • selector (Callable[IBaseData, float], optional) — The number of last stored indicator values
Return type:

RateOfChange
