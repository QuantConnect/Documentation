Creates a new T3MovingAverage indicator.

Parameters
Symbol symbol The symbol whose T3 we want.
Int32 period The period over which to compute the T3.
*Decimal volumeFactor (Optional) The volume factor to be used for the T3 (value must be in the [0,1] range, defaults to 0.7).
*Nullable<Resolution> resolution (Optional) The resolution.
*Func<IBaseData, Decimal> selector (Optional) Selects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Return

T3MovingAverage - The T3MovingAverage indicator for the requested symbol over the specified period.
