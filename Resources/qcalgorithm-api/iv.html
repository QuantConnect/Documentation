Creates a new ImpliedVolatility indicator for the symbol The indicator will be automatically updated on the symbol's subscription resolution.

Parameters
Symbol symbol The option symbol whose values we want as an indicator.
*Symbol mirrorOption (Optional) The mirror option contract used for parity type calculation.
*Nullable<Decimal> riskFreeRate (Optional) The risk free rate.
*Nullable<Decimal> dividendYield (Optional) The dividend yield.
*OptionPricingModelType optionModel (Optional) The option pricing model used to estimate IV.
*Int32 period (Optional) The lookback period of historical volatility.
*Nullable<Resolution> resolution (Optional) The desired resolution of the data.

Return

ImpliedVolatility - A new ImpliedVolatility indicator for the specified symbol.
