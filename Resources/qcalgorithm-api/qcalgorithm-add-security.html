add_security(security_type, ticker, resolution, market, fill_forward, leverage, extended_market_hours, data_mapping_mode=None, data_normalization_mode=None)[source]

Add specified data to our data subscriptions. QuantConnect will funnel this data to the handle data routine.

Parameters:
  • security_type (SecurityType) — MarketType Type: Equity, Commodity, Future, FOREX or Crypto
  • ticker (str) — The security ticker
  • resolution (Optional[Resolution]) — Resolution of the Data Required
  • market (str) — When no data available on a tradebar, return the last data that was generated
  • fill_forward (bool) — Custom leverage per security
  • leverage (float) — Use extended market hours data
  • extended_market_hours (bool) — The contract mapping mode to use for the security
  • data_mapping_mode (DataMappingMode, optional) — The price scaling mode to use for the security
  • data_normalization_mode (DataNormalizationMode, optional)
Return type:

Security
