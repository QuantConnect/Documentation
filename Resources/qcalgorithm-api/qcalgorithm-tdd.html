tdd(symbol, period, minimum_acceptable_return=0.0, resolution=None, selector=None)[source]

Creates a new TargetDownsideDeviation indicator. The target downside deviation is defined as the root-mean-square, or RMS, of the deviations of the realized return’s underperformance from the target return where all returns above the target return are treated as underperformance of 0.

Parameters:
  • symbol (Symbol) — The symbol for which the indicator is calculated.
  • period (int) — The period for Bollinger Bands.
  • minimum_acceptable_return (float, optional) — The multiplier for the Bollinger Bands' standard deviation.
  • resolution (Resolution, optional) — The period for Keltner Channels.
  • selector (Callable[IBaseData, float], optional) — The multiplier for the Average True Range in Keltner Channels.
Returns:

The configured Squeeze Momentum indicator.

Return type:

TargetDownsideDeviation
