Creates and adds a new Index security to the algorithm.

Parameters
String ticker The currency pair.
*Nullable<Resolution> resolution (Optional) The Resolution of market data, Tick, Second, Minute, Hour, or Daily. Default is Minute.
*String market (Optional) The index trading market, See also: Market. . Default value is None and looked up using BrokerageModel.DefaultMarkets in AddSecurity<T>.
*Boolean fillDataForward (Optional) If True, returns the last available data even if none in that timeslice. Default is True.

Return

Index - The new security.
