he(symbol, period, max_lag=20, resolution=None, selector=None)[source]

Creates a new Hurst Exponent indicator for the specified symbol. The Hurst Exponent measures the long-term memory or self-similarity in a time series. The default maxLag value of 20 is chosen for reliable and accurate results, but using a higher lag may reduce precision.

Parameters:
  • symbol (Symbol) — The symbol whose Heikin-Ashi we want
  • period (int) — The resolution
  • max_lag (int, optional) — Selects a value from the BaseData to send into the indicator, if null defaults to casting the input value to a TradeBar
  • resolution (Resolution, optional)
  • selector (Callable[IBaseData, float], optional)
Returns:

The Heikin-Ashi indicator for the requested symbol over the specified period

Return type:

HurstExponent
