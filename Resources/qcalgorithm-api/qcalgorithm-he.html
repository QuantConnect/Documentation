he(symbol, period, max_lag=20, resolution=None, selector=None)[source]

Creates a new Hurst Exponent indicator for the specified symbol. The Hurst Exponent measures the long-term memory or self-similarity in a time series. The default maxLag value of 20 is chosen for reliable and accurate results, but using a higher lag may reduce precision.

Parameters:
  • symbol (Symbol) — The option symbol whose values we want as an indicator
  • period (int) — The mirror option for parity calculation
  • max_lag (int, optional) — The risk free rate
  • resolution (Resolution, optional) — The dividend yield
  • selector (Callable[IBaseData, float], optional) — The option pricing model used to estimate Gamma
Returns:

A new Gamma indicator for the specified symbol

Return type:

HurstExponent
