            VolumeProfile QuantConnect.Algorithm.QCAlgorithm.VP (
    Symbol                            symbol,
    *Int32                            period,

Creates an Market Profile indicator for the symbol with Volume Profile (VOL) mode. The indicator will be automatically updated on the given resolution.

Parameters
Symbol symbol The symbol whose VP we want.
*Int32 period (Optional) The period of the VP.
*Decimal valueAreaVolumePercentage (Optional) The percentage of volume contained in the value area.
*Decimal priceRangeRoundOff (Optional) How many digits you want to round and the precision. i.e 0.01 round to two digits exactly.
*Resolution resolution (Optional) The resolution.
*Func<IBaseData, TradeBar> selector (Optional) Selects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Return

VolumeProfile - The Volume Profile indicator for the given parameters.
