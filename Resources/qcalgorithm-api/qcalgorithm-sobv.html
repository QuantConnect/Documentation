sobv(symbol, period, type=0, resolution=None, selector=None)[source]

Creates a new SmoothedOnBalanceVolume indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The symbol whose SAREXT we seek
  • period (int) — The starting value for the Stop and Reverse indicator
  • type (MovingAverageType, optional) — The offset value to be applied on reverse
  • resolution (Resolution, optional)
  • selector (Callable[IBaseData, IBaseDataBar], optional)
Return type:

SmoothedOnBalanceVolume
