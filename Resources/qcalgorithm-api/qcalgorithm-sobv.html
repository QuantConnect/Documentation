sobv(symbol, period, type=0, resolution=None, selector=None)[source]

Creates a new SmoothedOnBalanceVolume indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The symbol for the indicator to track
  • period (int) — The fast moving average period
  • type (MovingAverageType, optional) — The slow moving average period
  • resolution (Resolution, optional) — The signal period
  • selector (Callable[IBaseData, IBaseDataBar], optional) — The type of moving average to use
Returns:

The SchaffTrendCycle indicator for the requested symbol over the specified period
