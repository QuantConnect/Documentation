Creates a Alpha indicator for the given target symbol in relation with the reference used. The indicator will be automatically updated on the given resolution.

Parameters
Symbol target The target symbol whose Alpha value we want.
Symbol reference The reference symbol to compare with the target symbol.
*Int32 alphaPeriod (Optional) The period of the Alpha indicator.
*Int32 betaPeriod (Optional) The period of the Beta indicator.
*Nullable<Resolution> resolution (Optional) The resolution.
*Nullable<Decimal> riskFreeRate (Optional) The risk free rate.
*Func<IBaseData, IBaseDataBar> selector (Optional) Selects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Return

Alpha - The Alpha indicator for the given parameters.
