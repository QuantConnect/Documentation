sto(symbol, period, k_period, d_period, resolution=None, selector=None)[source]

Creates a new Stochastic indicator.

Parameters:
  • symbol (Symbol) — The symbol for which the indicator is calculated.
  • period (int) — The period for Bollinger Bands.
  • k_period (int) — The multiplier for the Bollinger Bands' standard deviation.
  • d_period (int) — The period for Keltner Channels.
  • resolution (Resolution, optional) — The multiplier for the Average True Range in Keltner Channels.
  • selector (Callable[IBaseData, TradeBar], optional) — The resolution of the data.
Returns:

The configured Squeeze Momentum indicator.

Return type:

Stochastic
