            OrderTicket QuantConnect.Algorithm.QCAlgorithm.TrailingStopOrder (
    Symbol             symbol,
    Int32              quantity,
    Decimal            trailingAmount,
    Boolean            trailingAsPercentage,
    *String            tag,
    *IOrderProperties  orderProperties
   )

Create a trailing stop order and return the newly created order id; or negative if the order is invalid. It will calculate the stop price using the trailing amount and the current market price.

Parameters
Symbol symbol Trading asset symbol.
Int32 quantity Quantity to be traded.
Decimal trailingAmount The trailing amount to be used to update the stop price.
Boolean trailingAsPercentage Whether the is a percentage or an absolute currency value.
*String tag (Optional) Optional string data tag for the order.
*IOrderProperties orderProperties (Optional) The order properties to use. Defaults to DefaultOrderProperties.

Return

OrderTicket - The order ticket instance.
