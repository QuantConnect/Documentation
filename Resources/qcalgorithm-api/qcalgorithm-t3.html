T3(symbol, period, volumeFactor=0.7, resolution=None, selector=None)[source]

Creates a new T3MovingAverage indicator.

  • symbol (Symbol) — The option symbol whose values we want as an indicator
  • period (Int32) — The mirror option for parity calculation
  • volumeFactor (decimal, optional) — The risk free rate
  • resolution (Resolution, optional) — The dividend yield
  • selector (Func<IBaseData, Decimal>, optional) — The option pricing model used to estimate Theta
Return type:

T3MovingAverage
