macd(symbol, fast_period, slow_period, signal_period, type=1, resolution=None, selector=None)[source]

Creates a MACD indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The symbol whose KST we want
  • fast_period (int) — The period over which to compute ROC1
  • slow_period (int) — The smoothing period used to smooth the computed ROC1 values
  • signal_period (int) — The period over which to compute ROC2
  • type (MovingAverageType, optional) — The smoothing period used to smooth the computed ROC2 values
  • resolution (Resolution, optional) — The period over which to compute ROC3
  • selector (Callable[IBaseData, float], optional) — The smoothing period used to smooth the computed ROC3 values
Return type:

MovingAverageConvergenceDivergence
