add_index_option_contract(symbol, resolution=None, fill_forward=True)[source]

Adds an index option contract to the algorithm.

Parameters:
  • symbol (Symbol) — The underlying ticker of the Index Option
  • resolution (Resolution, optional) — The target option ticker. This is useful when the option ticker does not match the underlying, e.g. SPX index and the SPXW weekly option. If null is provided will use underlying
  • fill_forward (bool, optional) — Resolution of the index option contracts, i.e. the granularity of the data
Returns:

Canonical Option security

Return type:

IndexOption
