mfi(symbol, period, resolution=None, selector=None)[source]

Creates a new MoneyFlowIndex indicator. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The symbol for which the MAMA indicator is being created.
  • period (int) — The fast limit for the adaptive moving average.
  • resolution (Resolution, optional) — The slow limit for the adaptive moving average.
  • selector (Callable[IBaseData, TradeBar], optional) — The resolution
Returns:

The Mesa Adaptive Moving Average (MAMA) indicator for the requested symbol with the specified limits.

Return type:

MoneyFlowIndex
