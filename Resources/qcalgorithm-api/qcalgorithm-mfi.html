mfi(symbol, period, resolution=None, selector=None)[source]

Creates a new MoneyFlowIndex indicator. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The symbol whose VP we want
  • period (int) — The period of the VP
  • resolution (Resolution, optional) — The percentage of volume contained in the value area
  • selector (Callable[IBaseData, TradeBar], optional) — How many digits you want to round and the precision. i.e 0.01 round to two digits exactly.
Returns:

The Volume Profile indicator for the given parameters

Return type:

MoneyFlowIndex
