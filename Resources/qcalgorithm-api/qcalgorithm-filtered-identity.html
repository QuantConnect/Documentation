filtered_identity(symbol, resolution, selector=None, filter=None, field_name=None)[source]

Creates a new FilteredIdentity indicator for the symbol The indicator will be automatically updated on the symbol's subscription resolution

  • symbol (Symbol) — The symbol whose EMV we want
  • resolution (Resolution | timedelta) — The period of the EMV
  • selector (Callable[IBaseData, IBaseDataBar] | PyObject, optional) — The length of the outputed value
  • filter (Callable[IBaseData, bool] | PyObject, optional) — The resolution
  • field_name (str, optional) — Selects a value from the BaseData to send into the indicator, if null defaults to casting the input value to a TradeBar
