dem(symbol, period, type, resolution=None, selector=None)[source]

Creates a new DeMarker Indicator (DEM), an oscillator-type indicator measuring changes in terms of an asset's High and Low tradebar values.

Parameters:
  • symbol (Symbol) — The symbol whose CRSI is to be calculated.
  • period (int) — The period for the traditional RSI calculation.
  • type (MovingAverageType) — The period for the Streak RSI calculation (SRSI).
  • resolution (Resolution, optional) — The look-back period for calculating the Percent Rank.
  • selector (Callable[IBaseData, TradeBar], optional) — The resolution of the data (optional).
Returns:

The Connors Relative Strength Index (CRSI) for the specified symbol and periods.

Return type:

DeMarkerIndicator
