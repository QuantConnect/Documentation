Creates and adds index options to the algorithm.

Parameters
String ticker The ticker of the Index Option.
*Nullable<Resolution> resolution (Optional) Resolution of the index option contracts, i.e. the granularity of the data.
*String market (Optional) Market of the index option. If no market is provided, we default to USA.
*Boolean fillDataForward (Optional) If True, this will fill in missing data points with the previous data point.

Return
