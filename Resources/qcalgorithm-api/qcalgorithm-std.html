std(symbol, period, resolution=None, selector=None)[source]

Creates a new StandardDeviation indicator. This will return the population standard deviation of samples over the specified period.

Parameters:
  • symbol (Symbol) — The symbol whose SAREXT we seek
  • period (int) — The starting value for the Stop and Reverse indicator
  • resolution (Resolution, optional) — The offset value to be applied on reverse
  • selector (Callable[IBaseData, float], optional) — The starting acceleration factor for short positions
Return type:

StandardDeviation
