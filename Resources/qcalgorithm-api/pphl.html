Creates a new PivotPointsHighLow indicator.

Parameters
Symbol symbol The symbol whose PPHL we seek.
Int32 lengthHigh The number of surrounding bars whose high values should be less than the current bar's for the bar high to be marked as high pivot point.
Int32 lengthLow The number of surrounding bars whose low values should be more than the current bar's for the bar low to be marked as low pivot point.
*Int32 lastStoredValues (Optional) The number of last stored indicator values.
*Nullable<Resolution> resolution (Optional) The resolution.
*Func<IBaseData, IBaseDataBar> selector (Optional) Selects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Return

PivotPointsHighLow - The PivotPointsHighLow indicator for the requested symbol.
