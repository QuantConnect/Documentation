Creates an FisherTransform indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters
Symbol symbol The symbol whose FisherTransform we want.
Int32 period The period of the FisherTransform.
*Nullable<Resolution> resolution (Optional) The resolution.
*Func<IBaseData, TradeBar> selector (Optional) Selects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Return

FisherTransform - The FisherTransform for the given parameters.
