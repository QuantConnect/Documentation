            TrueStrengthIndex QuantConnect.Algorithm.QCAlgorithm.TSI (
    Symbol                           symbol,
    *Int32                           longTermPeriod,
    *Int32                           shortTermPeriod,

    *MovingAverageType               signalType,
    *Nullable<Resolution>      resolution,

Creates a TrueStrengthIndex indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters
Symbol symbol The symbol whose TSI we want.
*Int32 longTermPeriod (Optional) Period used for the second (double) price change smoothing.
*Int32 shortTermPeriod (Optional) Period used for the first price change smoothing.
*Int32 signalPeriod (Optional) The signal period.
*MovingAverageType signalType (Optional) The type of moving average to use for the signal.
*Nullable<Resolution> resolution (Optional) The resolution.
*Func<IBaseData, Decimal> selector (Optional) Selects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Return

TrueStrengthIndex - The TrueStrengthIndex indicator for the given parameters.
