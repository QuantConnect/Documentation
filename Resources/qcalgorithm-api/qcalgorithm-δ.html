δ(symbol, mirror_option=None, risk_free_rate=None, dividend_yield=None, option_model=0, iv_model=None, resolution=None)[source]

Creates a new Delta indicator for the symbol The indicator will be automatically updated on the symbol's subscription resolution

Parameters:
  • symbol (Symbol) — The symbol whose Donchian Channel we seek.
  • mirror_option (Symbol, optional) — The period over which to compute the upper Donchian Channel.
  • risk_free_rate (float, optional) — The period over which to compute the lower Donchian Channel.
  • dividend_yield (float, optional) — The resolution.
  • option_model (OptionPricingModelType, optional) — Selects a value from the BaseData to send into the indicator, if null defaults to casting the input value to a TradeBar
  • iv_model (OptionPricingModelType, optional)
  • resolution (Resolution, optional)
Returns:

The Donchian Channel indicator for the requested symbol.

Return type:

Delta
