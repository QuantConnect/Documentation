vwma(symbol, period, resolution=None, selector=None)[source]

Creates a new VolumeWeightedMovingAverage indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The option symbol whose values we want as an indicator
  • period (int) — The mirror option for parity calculation
  • resolution (Resolution, optional) — The risk free rate
  • selector (Callable[IBaseData, TradeBar], optional) — The dividend yield
Return type:

VolumeWeightedMovingAverage
