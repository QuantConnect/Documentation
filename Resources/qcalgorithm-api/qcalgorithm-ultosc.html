ultosc(symbol, period_1, period_2, period_3, resolution=None, selector=None)[source]

Creates a new UltimateOscillator indicator.

Parameters:
  • symbol (Symbol) — The symbol whose TSI we want
  • period_1 (int) — Period used for the first price change smoothing
  • period_2 (int) — Period used for the second (double) price change smoothing
  • period_3 (int) — The signal period
  • resolution (Resolution, optional) — The type of moving average to use for the signal
  • selector (Callable[IBaseData, IBaseDataBar], optional) — The resolution
Returns:
