create_indicator_name(symbol, type, resolution)[source]

Creates a new name for an indicator created with the convenience functions (SMA, EMA, ect...)

Parameters:
  • symbol (Symbol) — The symbol for which to create the ZigZag indicator.
  • type (Formattablestr | str) — The sensitivity for detecting pivots.
  • resolution (Resolution) — The minimum number of bars required for a trend before a pivot is confirmed.
Return type:

str
