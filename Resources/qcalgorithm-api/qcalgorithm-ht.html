ht(symbol, length, in_phase_multiplication_factor, quadrature_multiplication_factor, resolution=None, selector=None)[source]

Creates a new Hilbert Transform indicator

Parameters:
  • symbol (Symbol) — The symbol for which the Hurst Exponent is calculated.
  • length (int) — The number of data points used to calculate the indicator at each step.
  • in_phase_multiplication_factor (float) — The maximum time lag used to compute the tau values for the Hurst Exponent calculation.
  • quadrature_multiplication_factor (float) — The resolution
  • resolution (Resolution, optional) — Function to select a value from the BaseData to input into the indicator. Defaults to using the 'Value' property of BaseData if null.
  • selector (Callable[IBaseData, float], optional)
Returns:

The Hurst Exponent indicator for the specified symbol.

Return type:

HilbertTransform
