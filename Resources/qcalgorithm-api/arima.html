Creates a new ARIMA indicator.

Parameters

Int32 diffOrder Difference order (d) -- defines how many times to difference the model before fitting parameters.
Int32 maOrder MA order (q) -- defines the number of past values to consider in the MA component of the model.
Int32 period Size of the rolling series to fit onto.
*Nullable<Resolution> resolution (Optional) The resolution.

Return
