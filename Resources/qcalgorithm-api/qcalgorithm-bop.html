bop(symbol, resolution=None, selector=None)[source]

Creates a new Balance Of Power indicator. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The symbol whose BollingerBands we seek
  • resolution (Resolution, optional) — The period of the standard deviation and moving average (middle band)
  • selector (Callable[IBaseData, IBaseDataBar], optional) — The number of standard deviations specifying the distance between the middle band and upper or lower bands
Returns:

A BollingerBands configured with the specified period

Return type:

BalanceOfPower
