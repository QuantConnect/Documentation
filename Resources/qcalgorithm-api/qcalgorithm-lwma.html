lwma(symbol, period, resolution=None, selector=None)[source]

Creates a new LinearWeightedMovingAverage indicator. This indicator will linearly distribute the weights across the periods.

Parameters:
  • symbol (Symbol) — The symbol whose KST we want
  • period (int) — The period over which to compute ROC1
  • resolution (Resolution, optional) — The smoothing period used to smooth the computed ROC1 values
  • selector (Callable[IBaseData, float], optional) — The period over which to compute ROC2
Returns:

A new KnowSureThing indicator with the specified smoothing type and period

Return type:
