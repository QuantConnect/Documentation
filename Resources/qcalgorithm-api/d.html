Creates a new Delta indicator for the symbol The indicator will be automatically updated on the symbol's subscription resolution.

Parameters
Symbol symbol The option symbol whose values we want as an indicator.
*Symbol mirrorOption (Optional) The mirror option for parity calculation.
*Nullable<Decimal> riskFreeRate (Optional) The risk free rate.
*Nullable<Decimal> dividendYield (Optional) The dividend yield.
*OptionPricingModelType optionModel (Optional) The option pricing model used to estimate Delta. Options: ['BlackScholes', 'BinomialCoxRossRubinstein', 'ForwardTree']
*Nullable<OptionPricingModelType> ivModel (Optional) The option pricing model used to estimate IV.
*Nullable<Resolution> resolution (Optional) The desired resolution of the data.

Return

Delta - A new Delta indicator for the specified symbol.
