LEAN currently supports over 100 indicators. This page explains how to contribute a new indicator to the open-source project by making a pull request to Lean. Before you get started, familiarize yourself with our contributing guidelines, and review previous contributions to understand our standards. If you don't already have a new indicator in mind that you want to contribute, see the GitHub Issues in the Lean repository for a list of indicators that community members have requested. Before you begin, create a GitHub Issue with a description of your proposed indicator, links to relevant resources, and source of third-party values.
