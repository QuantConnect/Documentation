The following table contains the details of all configuration keys.

Key Description Type Default Value
environment The environment that the LEAN engine should be running at.
Options:
  • live-paper
  • backtesting
  • live-interactive
  • live-interactive-iqfeed
 string "backtesting"
algorithm-type-name The algorithm class to be backtested or live traded. string "BasicTemplateFrameworkAlgorithm"
algorithm-language The programming language of the algorithm class.
Options:
  • CSharp
  • Python
 string "CSharp"
algorithm-location The path of the algorithm class. Usually it is "<namespace>.dll" for CSharp, and "../../../Algorithm.Python/<filename>.py". string "QuantConnect.Algorithm.CSharp.dll"
data-folder The folder contains LEAN historical data. string "../../../Data/"
debugging If running in debugging mode or not. bool false
debugging-method The mode/method of the debugger.
Options:
  • LocalCmdLine
  • VisualStudio
  • Debugpy
  • PyCharm
 string "LocalCmdline"
log-handler The class of the log handler. string "QuantConnect.Logging.CompositeLogHandler"
messaging-handler The class of the messaging handler. string "QuantConnect.Messaging.Messaging"
job-queue-handler The class of the job queue handler. string "QuantConnect.Queues.JobQueue"
api-handler The class of the API handler. string "QuantConnect.Api.Api"
map-file-provider The class of the asset map file provider. string "QuantConnect.Data.Auxiliary.LocalDiskMapFileProvider"
factor-file-provider The class of the asset factor file provider. string "QuantConnect.Data.Auxiliary.LocalDiskFactorFileProvider"
data-provider The class of the data provider. string "QuantConnect.Lean.Engine.DataFeeds.DefaultDataProvider"
alpha-provider The class of the alpha provider. string "QuantConnect.Lean.Engine.Alphas.DefaultAlphaHandler"
data-channel-provider The class of the data channel provider. string "DataChannelProvider"
object-store The class of the local object store handler. string "QuantConnect.Lean.Engine.Storage.LocalObjectStore"
data-aggregator The class of the data aggregation handler. string "QuantConnect.Lean.Engine.DataFeeds.AggregationManager"
symbol-minute-limit The maximum number of symbol allowed for minute resolution data. int 10000
symbol-second-limit The maximum number of symbol allowed for second resolution data. int 10000
symbol-tick-limit The maximum number of symbol allowed for tick resolution data. int 10000
maximum-data-points-per-chart-series The maximum number of data point for each series in a chart. int 4000
force-exchange-always-open If the market hours will remain open all the time or not. bool false
transaction-log The path of the CSV file to output the list of transactions if needed. In default, it will not save one. string ""
job-user-id The user ID of QuantConnect API. In default, it will not link to QC cloud. Click here to obtain the API user ID and token. string "0"
api-access-token The access token of QuantConnect API. In default, it will not link to QC cloud. Click here to obtain the API user ID and token. string ""
job-organization-id The organization ID in QuantConnect. In default, it will not link to QC cloud. string ""
live-data-url The URL of which the live data is streaming. In default, it uses QuantConnect as live data provider. string "ws://www.quantconnect.com/api/v2/live/data/"
live-data-url The port of which the live data is streaming. int 8020
live-data-url The initial cash balance of the live portfolio. Only used in QC paper trading. string ""
live-holdings The initial security holdings of the live portfolio. Only used in QC paper trading. string "[]"
parameters The parameters to set in the algorithm class. Dictionary
