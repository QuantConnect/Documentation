The Alpha Streams software development kit (SDK) has a REST client that enables you to perform the following actions:

  • Request details about a specific alpha
  • Get information about a specific author
  • Get the errors for a specific alpha
  • Get the equity curve of a specific alpha
  • Get a list of all alpha IDs
  • Get the orders and OrderEvents for a specific alpha
  • Get the current holdings of a specific alpha
  • Download all orders of an alpha and save them to file
  • Search for alpha that match specific criteria
  • Search for authors that match specific criteria
  • Create and remove bids for alphas
  • Get details about a bid for a specific alpha

For more information on the Alpha Streams SDK, refer to the Alpha Streams documentation.
