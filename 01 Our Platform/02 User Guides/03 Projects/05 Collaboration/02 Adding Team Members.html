Add team members to your project to simultaneously work on the same code files together. When there are multiple people working on the same code file, the cursor of each member is visible in the IDE and all file changes occur in real-time for everyone. All of the team members that you add to a project have access to the project from their QuantConnect account. The number of members that you can add to a project is limited by the tier of your organization. The limits of each tier are the following:

  • Free: Unsupported
  • Quant Researcher: Unsupported
  • Team: 10
  • Trading Firm: Unlimited
  • Institution: Unlimited
