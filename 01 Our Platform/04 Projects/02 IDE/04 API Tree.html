The API Tree is a panel in the IDE that describes all of the members of the QCAlgorithm class. The type and description are included for class attributes. The description, argument types, argument names, and return type are included for class methods. The API Tree uses two icons to distinguish between class properties and methods:

  • Wrench: Property
  • Cube: Method

Use the API Tree to increase your development speed. You can find which class members you have available to use and what arguments they may expect without needing to reference the documentation.
