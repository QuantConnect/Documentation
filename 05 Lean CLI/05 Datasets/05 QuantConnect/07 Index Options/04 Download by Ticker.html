To download data for selected tickers instead of the full dataset, run a non-interactive lean data download command. First, download the prerequisite US Index Option Universe:

$ lean data download --dataset "US Index Option Universe" --data-type "bulk" --start "20250403" --end "20250403"

Then download the US Index Options data for the tickers, resolution, and date range you need. For example, to download minute-resolution trade data for VIX:

$ lean data download --dataset "US Index Options" --data-type "Trade" --ticker "VIX" --resolution "Minute" --start "20230101" --end "20230105"

To download data interactively or to use the CLI Command Generator, see Using the CLI.
