To download data for selected tickers instead of the full dataset, run a non-interactive lean data download command. For example, to download minute-resolution quote data for USDCAD:

$ lean data download --dataset "FOREX Data" --data-type "Quote" --ticker "USDCAD" --resolution "Minute" --start "20230101" --end "20230105"

To download data interactively or to use the CLI Command Generator, see Using the CLI.
