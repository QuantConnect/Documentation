You can acquire local data in bulk or by ticker. The method you choose affects how you pay for the data and which data types you receive.

Bulk

When you download a dataset in bulk, you subscribe to it and download the full dataset with daily updates for one year. After that year, subscribe to the cheaper updates package to keep your local copy current for another year. Bulk downloads include every data type the dataset provides.

By Ticker

When you download by ticker, you download only the tickers you select and pay QuantConnect Credit (QCC) per file. Some datasets provide up to three data types: Trade, Quote, and Open Interest. These data types are not mandatory, but to reproduce the same results as QuantConnect Cloud, download all the data types that the dataset provides.
