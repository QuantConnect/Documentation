To download data for selected tickers instead of the full dataset, run a non-interactive lean data download command. First, download the prerequisite datasets:

$ lean data download --dataset "US Equity Security Master"
$ lean data download --dataset "US Equity Option Universe" --data-type "bulk" --start "20250403" --end "20250403"

Then download the US Equity Options data for the tickers, resolution, and date range you need. For example, to download minute-resolution trade data for GOOG:

$ lean data download --dataset "US Equity Options" --data-type "Trade" --ticker "GOOG" --resolution "Minute" --start "20230101" --end "20230105"

You also need data for the underlying Equity; see US Equity. To download data interactively or to use the CLI Command Generator, see Using the CLI.
