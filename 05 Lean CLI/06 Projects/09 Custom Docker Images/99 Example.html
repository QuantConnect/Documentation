In this following example, we will demonstate how to use Julia in python's environment via a custom docker image.

  1. Follow step 1-2 in the Building Custom Images tutorial to obtain the basic Lean foundation.
  2. Add the following instructions to install Julia in the docker file DockerfileLeanFoundation of Lean.
    3. # Install Julia
RUN wget -q https://julialang-s3.julialang.org/bin/linux/x64/1.8/julia-1.8.5-linux-x86_64.tar.gz && \
    tar zxvf julia-1.8.5-linux-x86_64.tar.gz && \
    rm julia-1.8.5-linux-x86_64.tar.gz
ENV PATH="/julia-1.8.5/bin:${PATH}"

# Install Julia package
RUN pip install --no-cache-dir julia==0.6.0

# Install Julia so it's ready for usage
RUN python -m julia.sysimage sys.so
  3. Continue to build the custom docker image following step 4-5 in the Building Custom Images tutorial.

After building the Julia-compatible image, you are able to code with Julia under PyJulia interface.

from julia import Julia
jl = Julia(sysimage=""/sys.so"")
from julia import Base

Base.sind(90)
