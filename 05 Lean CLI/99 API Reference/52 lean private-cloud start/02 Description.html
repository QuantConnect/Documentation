Starts a new private cloud deployment in a Docker container. Run with --master to start the node as a master server, or --slave to start it as a slave node connected to an existing master. Use --token, --master-domain, and --master-port to configure the connection between master and slave nodes.

Use --compute to select a compute configuration and --extra-docker-config to pass additional Docker configuration as a JSON string. Pass --update to pull the latest Docker image before starting, or --stop to stop any existing deployment first.
