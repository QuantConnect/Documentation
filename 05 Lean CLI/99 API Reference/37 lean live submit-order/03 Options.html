The lean live submit-order command supports the following options:

Option Description
--ticker TEXT [required]
--market TEXT [required]
--security-type TEXT [required]
--order-type TEXT [required]
--quantity DECIMAL [required]
--limit-price DECIMAL The limit price of the order be submitted
--stop-price DECIMAL The stop price of the order to be submitted
--tag TEXT The tag to be attached to the order
--lean-config FILE The Lean configuration file that should be used (defaults to the nearest lean.json)
--verbose Enable debug logging
--help Show this message and exit.
