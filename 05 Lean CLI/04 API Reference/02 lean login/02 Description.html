Lets you log in with your QuantConnect API credentials and stores the given values in the credentials file in your global configuration directory.

If --user-id or --api-token is not provided an interactive prompt is shown and the missing values are read from stdin.

You can request your user id and API token by clicking on the "Request Email With Token and Your User-Id for API Requests" link on your QuantConnect Account page. After clicking on this link your user id and API token are automatically sent to your email address.
