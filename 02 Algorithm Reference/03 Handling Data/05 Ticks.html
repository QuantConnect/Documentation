$[Ticks,T:QuantConnect.Data.Market.Tick] data provides LastPrice and Quantity properties for a given time. If it is a quote tick, it also contains non-zero BidPrice, BidSize, AskPrice, AskSize properties. A Trade Tick is a record of a transaction or sale for the security. A Quote Tick is a bid or offer to purchase the security for a specific price. 

    self.AddEquity("IBM", Resolution.Tick) ## Subscribe to tick-level IBM data
        
def OnData(self, data):
     ## Ticks can get pushed together and arrive at the same time
     ## and have the same EndTime in the data handler, so we use 
     ## the [-1] to access to most recent tick that arrived
     
     self.Log('Last price: ' + str(data['IBM'][-1].LastPrice))
     self.Log('Quantity: ' + str(data['IBM'][-1].Quantity))
