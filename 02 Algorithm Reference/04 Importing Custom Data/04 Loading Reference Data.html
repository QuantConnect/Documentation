This is possible with the $[Download(),M:QuantConnect.Algorithm.QCAlgorithm.Download] method. The Download method downloads a URL and returns it as a string. It can take header settings for authentication, and a username and password for basic authentication.

It is a common request to download data from a public Dropbox file. In this case, you should ensure you're downloading the direct file link - not the HTML page of the download. You can specify this by adding &dl=1 to the end of the Dropbox download URL.
