QuantConnect hosts transactions and intentions data for U.S. Equity markets provided by Smart Insider.

To import the $[SmartInsiderTransaction,T:QuantConnect.Data.Custom.SmartInsider] and $[SmartInsiderIntention,T:QuantConnect.Data.Custom.SmartInsider] types into your algorithm, you can do the following: 

using QuantConnect.Data.Custom.SmartInsider;
        
from QuantConnect.Data.Custom.SmartInsider import *

You can add this data to your algorithm by using AddData: 

AddData<SmartInsiderTransaction>("AAPL", Resolution.Daily, TimeZones.Utc);
AddData<SmartInsiderIntention>("AAPL", Resolution.Daily, TimeZones.Utc);
        
AddData(SmartInsiderTransaction, "AAPL", Resolution.Daily, TimeZones.Utc)
AddData(SmartInsiderIntention, "AAPL", Resolution.Daily, TimeZones.Utc)
