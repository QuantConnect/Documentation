QuantConnect provides PsychSignal sentiment data for use in your algorithms. PsychSignal sentiment data is based on user sentiment from both Twitter and StockTwits.

The data begins on 2010-01-01 05:00:00 UTC, and continues to the present day.

To add PsychSignal data to your algorithm, use the type $[PsychSignalSentiment,T:QuantConnect.Data.Custom.PsychSignal].
