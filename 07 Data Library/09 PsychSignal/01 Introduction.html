QuantConnect provides PsychSignal sentiment data for use in your algorithms. PsychSignal sentiment data is based on user sentiment from both Twitter and StockTwits.

The data begins on 2010-01-01 05:00:00 UTC, and continues to the present day.

To import the type to the algorithm, you can do the following: 

using QuantConnect.Data.Custom.PsychSignal;
        
from QuantConnect.Data.Custom.PsychSignal import *

To add PsychSignal data to your algorithm, use the type $[PsychSignalSentimentData,T:QuantConnect.Data.Custom.PsychSignal] when adding data with AddData: 

// Adding PsychSignal sentiment data to the algorithm
AddData<PsychSignalSentimentData>("AAPL", Resolution.Minute, TimeZones.Utc);
        
# Adding PsychSignal sentiment data to the algorithm
self.AddData(PsychSignalSentimentData, "AAPL", Resolution.Minute, TimeZones.Utc)
