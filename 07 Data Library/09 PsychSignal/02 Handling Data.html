PsychSignal data only supports the resolution Resolution.Minute. To construct Resolution.Hour and Resolution.Daily, consider using Consolidators

$[PsychSignalSentiment,T:QuantConnect.Data.Custom.PsychSignal] has the following fields available:

Field Type Description
BullIntensity decimal Bullish (positive) intensity of messages scanned at that minute. This is a positive value between 0.0 and 4.0 (inclusive)
BearIntensity decimal Bearish (negative) intensity of messages scanned at that minute. This is a positive value between 0.0 and 4.0 (inclusive)
BullMinusBear decimal Bullish intensity minus bearish intensity
BullScoredMessages int Number of bullish messages scanned at that minute. Note that BullScoredMessages + BearScoredMessages can be smaller than TotalScoredMessages due to the sentiment classification algorithm not being able to classify the message as bullish or bearish
BearScoredMessages int Number of bearish messages scanned at that minute. Note that BullScoredMessages + BearScoredMessages can be smaller than TotalScoredMessages due to the sentiment classification algorithm not being able to classify the message as bullish or bearish
BullBearMessageRatio decimal BullScoredMessages / BearScoredMessages - If BearScoredMessages is zero, then the ratio will be reported as 0
TotalScoredMessages int Total messages scanned related to the ticker. This can be greater than BullScoredMessages + BearScoredMessages because the sentiment classification algorithm was unable to classify a message into bullish or bearish

PsychSignal data is prone to rename and delisting events. This is because the custom data is related to equity symbols, which can undergo rename and delisting events

Specify the timezone as UTC in AddData
