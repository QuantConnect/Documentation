QuantConnect sources yield curve data from the United States Treasury (U.S. Treasury) at the end of each trading day. To access the data, use the type: $[USTreasuryYieldCurve,T:QuantConnect.Data.Custom.USTreasury].

To import the type into your algorithm, you can do the following: 

using QuantConnect.Data.Custom.USTreasury;
        
from QuantConnect.Data.Custom.USTreasury import *

To add the data to your algorithm, use AddData: 

// US Treasury yield curve data has no associated ticker. Use a placeholder value instead
AddData<USTreasuryYieldCurve>("FOOBAR");
