U.S. Treasury yield curve data has no associated ticker with it. To add it to your algorithm, use a placeholder value. The Symbol object for the U.S. Treasury custom data is assigned the ticker/value you pass into AddData, and will never undergo delisting/rename events. An example of adding the data is provided: 

public class YieldCurveAlgorithm : QCAlgorithm 
{
    public void Initialize() 
    {
        AddData<USTreasuryYieldCurveRate>("FOOBAR", Resolution.Daily);
    }
}
        
class YieldCurveAlgorithm(QCAlgorithm):
    def Initialize(self):
        self.AddData(USTreasuryYieldCurveRate, "FOOBAR", Resolution.Daily)

All fields with the exception of BaseData fields contained within $[USTreasuryYieldCurveRate,T:QuantConnect.Data.Custom.USTreasury] can potentially be null values.
