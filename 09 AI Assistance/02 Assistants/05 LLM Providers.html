By default, Assistants run using our free QuantConnect Meta Agent. It requires no setup and is the right choice when you want the assistant working without bringing your own LLM API key.

Organizations with a paid Assistant Node can supply an API key from an LLM provider of their choice. With your own provider, you can select a specific model, configure the reasoning effort, the maximum number of turns the assistant takes per deployment, and the tool selection policy. For the setup steps and how to get a key from each provider, see Bring Your Own Key.
