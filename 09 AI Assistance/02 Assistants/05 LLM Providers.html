By default, Assistants run using our free QuantConnect Meta Agent. It requires no setup and is the right choice when you want the assistant working without bringing your own LLM API key.

Organizations on a paid tier can supply an API key from an LLM provider of their choice. With your own provider, you can select a specific model, configure the reasoning effort, the maximum number of turns the assistant takes per deployment, and the tool selection policy. Use this option when you want to swap in a specific model, tune those parameters, or route costs through your own account.

Configure Your Own Provider

In the Settings section of the assistant, under Provider, select Use Your Own Provider. The following settings become available:

  • Provider: The LLM provider to use. The available options are anthropic, deepseek, google, grok, and openai.
  • API Token: The API key from your provider account.
  • Model: The model to run. The list shows the models that the selected provider offers.
  • Reasoning Effort: The amount of reasoning the model applies before it responds. The default is high.
  • Max Turns: The maximum number of turns the assistant takes per deployment. The default is 100.
  • Tool Choice: The tool selection policy. The default, auto, lets the model decide when to call tools.

Get an API Key

Each provider issues API keys from its developer console:

  • Anthropic: Log in or create an account on the Claude Console. Then, on the API Keys page, click Create Key.
  • DeepSeek: Log in or create an account on the DeepSeek Platform. Then, on the API Keys page, click Create new API key.
  • Google: Sign in with your Google account to Google AI Studio. Then, on the API Keys page, click Create API key.
  • Grok: Log in or create an account on the xAI Console. Then, on the API Keys page, click Create API key.
  • OpenAI: Log in or create an account on the OpenAI Platform. Then, on the API Keys page, click Create new secret key.

The free A-MICRO Assistant Node keeps its 100,000 tokens per month cap even when you use your own provider.
