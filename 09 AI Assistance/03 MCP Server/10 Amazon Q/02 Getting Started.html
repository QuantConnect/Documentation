To connect Amazon Q in Local Platform to the QC MCP Server, follow these steps:

  1. In Local Platform, install the Amazon Q extension.
  2. In the left navigation menu of Local Platform, right-click the Amazon Q icon and then click Move To > Secondary Side Bar.

    3. This layout enables you to view the QuantConnect Project panel and the Amazon Q panel at the same time.

  3. Edit the ~ / .aws / amazonq / mcp.json file on your machine to be the following configuration:
    4. {
  "useLegacyMcpJson": true,
  "mcpServers": {
    "qc-mcp": {
      "type": "http",
      "url": "http://localhost:3001/",
      "disabled": false
    }
  },
  "name": "q_ide_default",
  "prompt": "",
  "toolAliases": {}
}
  4. Create a new project or open an existing one.
  5. In the top navigation bar, click View > Chat.
  6. In the Chat panel, click the Amazon Q tab.
  7. In the Chat section, click the Configure MCP servers icon.
  8. In the MCP servers section, click the Refresh MCP servers icon.

    9. The QC MCP server should start up and show that tools are available.

  9. Click the close icon to return to the chat area.
