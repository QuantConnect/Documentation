To connect the Cascade agent in the Windsurf Editor to the QC MCP Server, follow these steps:

  1. Install and open the Windsurf Editor.
  2. In the top navigation bar of Windsurf, click View > Extensions.
  3. In the Search Exensions in Marketplace field, enter QuantConnect.
  4. Click Install.
  5. In your ~ / .codeium / windsurf / mcp_config.json file, add the following configuration:
    6. {
  "mcpServers": {
    "qc-mcp": {
      "disabled": false,
      "url": "http://localhost:3001/"
    }
  }
}
  6. In Windsurf, create a new project or open an existing one.
  7. In the top navigation menu, click File > Preferences > Windsurf Settings.
  8. On the Windsurf Settings page, in the Cascade section, click Open MCP Marketplace.
  9. If the QC MCP Server has an error, click the refresh icon to restart it.
  10. Press Ctrl+L to open Cascade and begin interacting with the agent.
