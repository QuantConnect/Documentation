To connect Claude Code in Local Platform to the QC MCP Server, follow these steps:

  1. In Local Platform, install the Claude Code extension.
  2. In your ~ / .claude.json file, add the following configuration:
    3. "mcpServers": {
  "qc-mcp": {
    "type": "http",
    "url": "http://localhost:3001/"
  }
},
  3. Create a new project or open an existing one.
  4. In the top navigation bar, click the Claude Code icon.
  5. In the Claude Code panel, enter / and then click MCP status from the menu.

    6. After about 10 seconds, it should show the MCP server is connected.

  6. If the MCP server doesn't connect after 10 seconds, close the Claude Code panel and re-open a new one.
