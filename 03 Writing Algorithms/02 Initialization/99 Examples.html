Example 1: Set Data Seed

Some data might not have data feed on its first fetch, making the algorithm prone to error. To solve this case, you can set a seed on using the last known price as the initial data.

public override void Initialize()
{
    SetSecurityInitializer(new BrokerageModelSecurityInitializer(
        BrokerageModel, new FuncSecuritySeeder(GetLastKnownPrices)));
}
def initialize(self):
    self.set_security_initializer(BrokerageModelSecurityInitializer(
        self.brokerage_model, FuncSecuritySeeder(self.get_last_known_prices)))

Example 2: Extended Market Hour

Some traders would use pre-market hour data for filtering assets. To do so, you can subscribe to extended market hour data with the following code:

public override void Initialize()
{
    UniverseSettings.ExtendedMarketHours = true;
}
def initialize(self):
    self.universe_settings.extended_market_hours = True

Example 3: Train Model On Warm Up Finishes

After warming up indicators or rolling windows, you can train machine learning model with the cached data:

public override void OnWarmupFinished()
{
    Train(MyTrainingMethod);
}
def on_warmup_finished(self) -> None:
    self.train(self.my_training_method)
