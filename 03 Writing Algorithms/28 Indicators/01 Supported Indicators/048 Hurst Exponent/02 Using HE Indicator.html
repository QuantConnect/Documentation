To create an automatic indicator for HurstExponent, call the HEhe helper method from the QCAlgorithm class. The HEhe method creates a HurstExponent object, hooks it up for automatic updates, and returns it so you can used it in your algorithm. In most cases, you should call the helper method in the Initializeinitialize method.

public class HurstExponentAlgorithm : QCAlgorithm
{
    private Symbol _symbol;
    private HurstExponent _he;

    public override void Initialize()
    {
        _symbol = AddEquity("SPY", Resolution.Daily).Symbol;
        _he = HE(_symbol, 32);
    }

    public override void OnData(Slice data)
    {

        if (_he.IsReady)
        {
            // The current value of _he is represented by itself (_he)
            // or _he.Current.Value
            Plot("HurstExponent", "he", _he);
        }
    }
}

You can manually create a HurstExponent indicator, so it doesn't automatically update. Manual indicators let you update their values with any data you choose.

Updating your indicator manually enables you to control when the indicator is updated and what data you use to update it. To manually update the indicator, call the Updateupdate method. The indicator will only be ready after you prime it with enough data.

public class HurstExponentAlgorithm : QCAlgorithm
{
    private Symbol _symbol;
    private HurstExponent _hurstexponent;

    public override void Initialize()
    {
        _symbol = AddEquity("SPY", Resolution.Daily).Symbol;
        _hurstexponent = new HurstExponent(32);
    }

    public override void OnData(Slice data)
    {
        if (data.Bars.TryGetValue(_symbol, out var bar))
            _hurstexponent.Update(bar.EndTime, bar.Close);

        if (_hurstexponent.IsReady)
        {
            // The current value of _hurstexponent is represented by itself (_hurstexponent)
            // or _hurstexponent.Current.Value
            Plot("HurstExponent", "hurstexponent", _hurstexponent);
        }
    }
}
class HurstExponentAlgorithm(QCAlgorithm):
    def initialize(self) -> None:
        self._symbol = self.add_equity("SPY", Resolution.DAILY).symbol
        self._hurstexponent = HurstExponent(32)

    def on_data(self, slice: Slice) -> None:
        bar = slice.bars.get(self._symbol)
        if bar:
            self._hurstexponent.update(bar.end_time, bar.close)

        if self._hurstexponent.is_ready:
            # The current value of self._hurstexponent is represented by self._hurstexponent.current.value
            self.plot("HurstExponent", "hurstexponent", self._hurstexponent.current.value)
