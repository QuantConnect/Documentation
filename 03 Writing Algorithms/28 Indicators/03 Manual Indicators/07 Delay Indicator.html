The Delay indicator delays its input by a specified number of bars. It is mainly used to combine indicators, enabling you to set a delay without using built-in rolling windows. For example, you can create a displaced moving average by applying a Delay to a SimpleMovingAverage. 

// Create a 20-bar Delay indicator
private Delay _delay;
_delay = new Delay(20);

// Create a displaced SMA by combining Delay with SMA
var sma = new SimpleMovingAverage(10);
var displacedSma = new Delay(5).Of(sma);
# Create a 20-bar Delay indicator
self.delay = Delay(20)

# Create a displaced SMA by combining Delay with SMA
sma = SimpleMovingAverage(10)
displaced_sma = IndicatorExtensions.of(Delay(5), sma)

For a full example of using the Delay indicator to create a displaced SMA ribbon strategy, see Combining Indicators Examples.
