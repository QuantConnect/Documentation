The process to update custom indicators is the same process you use to update manual indicators. For more information about updating manual indicators, see Manual Updates or Automatic Updates.

The process to update custom indicators is the same process you use to update manual indicators. If you update your custom indicator manually, you must set the current attribute, define the is_ready property, and call the on_updated method in the update method to trigger update events. When you register your custom indicator for automatic updates, these attributes are set automatically.
