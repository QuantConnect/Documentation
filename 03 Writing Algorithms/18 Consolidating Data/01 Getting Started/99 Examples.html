Example 1: 5-minute Bar

Many traders would use 5-minute trade bar to obtain sufficient OHLCV information for analysis. We can set up a time consolidator in this case.

var fiveMinuteConsolidator = new TradeBarConsolidator(TimeSpan.FromMinutes(5));
five_minute_consolidator = TradeBarConsolidator(timedelta(minutes=5))

Example 2: Dollar Bar

Financial machine learning expert Marcos Lopez de Prado suggested that using dollar bar would provide better information than time bar due to fairness and variation in machine learning.

var dollarBarConsolidator = new ClassicRenkoConsolidator(1);
dollar_bar_consolidator = ClassicRenkoConsolidator(1)

Example 3: Timestamp Bar

Some products have long trading time and special daily roll time, such as 17pm for Metals Futures. We can set up a daily trade bar that starts at that time of the previous day.

var metalFutureDailyConsolidator = new TradeBarConsolidator(datetime => {
    var period = TimeSpan.FromDays(1);
    var timeSpan = new TimeSpan(17, 0, 0);
    var newDateTime = datetime.Date + timeSpan;
    return new CalendarInfo(newDateTime.AddDays(-1), period);
});
metal_future_daily_consolidator = TradeBarConsolidator(lambda dt: CalendarInfo(dt.replace(hour=17, minute=0, second=0, microsecond=0) - timedelta(1), timedelta(1)))
