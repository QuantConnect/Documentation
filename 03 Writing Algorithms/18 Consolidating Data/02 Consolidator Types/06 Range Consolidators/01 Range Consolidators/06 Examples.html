Example 1: High Price Range Bar

Using range bar with high price can capture the fluctuation of high price levels for ultra short-term reversal strategy.

var highRangeConsolidator = new RangeConsolidator(1, data => (data as TradeBar).High);
high_range_consolidator = RangeConsolidator(1, lambda data: data.high)

Example 2: Volume Range Bar

Volume Range bar can provide insight for analysis in capital flow direction and rate.

var volumeRangeConsolidator = new RangeConsolidator(1000, data => (data as TradeBar).Volume);
volume_range_consolidator = RenkoConsolidator(1000, selector = lambda data: data.volume)

Example 3: Bid-Ask Spread Bar

A spread bar is an alternative measure of the bid-ask spread for prediction of liquidity.

var bidAskSpreadConsolidator = new RangeConsolidator(0.1, data => 
{
    var bar = data as QuoteBar;
    return bar.Ask.Close - bar.Bid.Close;
})
bid_ask_spread_consolidator = RangeConsolidator(0.1, selector = lambda data: data.ask.close - data.bid.close)
