Example 1: 5-minute Bar

Many traders would use 5-minute trade bar to obtain sufficient OHLCV information for analysis. We can set up a time consolidator in this case.

var fiveMinuteConsolidator = new TradeBarConsolidator(TimeSpan.FromMinutes(5));
five_minute_consolidator = TradeBarConsolidator(timedelta(minutes=5))

Example 2: 4-hour Bar

Some traders would use a "half-day" bar to feed to indicators for technical analysis.

var fourHourConsolidator = new TradeBarConsolidator(TimeSpan.FromHours(4));
four_hour_consolidator = TradeBarConsolidator(timedelta(hours=4))

Example 3: Ultra Short-term Quote Bar

To facilitate execution performance, ultra short-term quotebars and quote ticks might be used to analysis market depth.

var shortTermQuoteConsolidator = new TickQuoteBarConsolidator(TimeSpan.FromMilliseconds(100));
short_term_quote_consolidator = TickQuoteBarConsolidator(timedelta(milliseconds=100))
