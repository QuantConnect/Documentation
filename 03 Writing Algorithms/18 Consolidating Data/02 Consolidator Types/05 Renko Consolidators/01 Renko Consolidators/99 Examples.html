dollar_bar_consolidator = RenkoConsolidator(1)

Example 2: High-Low Range Bar

You can also use Renko bars with alternative measurements other than price. One example is that you can create High - Low range bars for prediction of volatility.

var highLowRangeConsolidator = new RenkoConsolidator(10, data => 
{
    var bar = data as TradeBar;
    return bar.High - bar.Low;
});
high_low_range_consolidator = RenkoConsolidator(10, selector = lambda data: data.high - data.low)

Example 3: Bid-Ask Spread Bar

Like example 2, a spread bar is an alternative measure of the bid-ask spread for prediction of liquidity.

var bidAskSpreadConsolidator = new RenkoConsolidator(0.05, data => 
{
    var bar = data as QuoteBar;
    return bar.Ask.Close - bar.Bid.Close;
});
bid_ask_spread_consolidator = RenkoConsolidator(0.05, selector = lambda data: data.ask.close - data.bid.close)
