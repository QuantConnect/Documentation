Except ordinary volume renko bars, you can also use create volume renko bars with ordinary renko consolidator using volume alternative measurements.

Example 1: Bid Volume Bar

Volume bar that only accepts bid volume to study the market demand.

var bidVolumeConsolidator = new RenkoConsolidator(1000, data => 
{
    var bar = data as QuoteBar;
    return bar.BidSize;
});
bid_volume_consolidator = RenkoConsolidator(1000, lambda data: bar.bid_size)

Example 2: Volume Direction

We can train model to predict the capital flow using a volume renko bar of the difference of ask volume versus bid volume.

var volumeDirectionConsolidator = new RenkoConsolidator(1000, data =>
{
    var bar = data as QuoteBar;
    return bar.AskSize - bar.BidSize;
});
volume_direction_consolidator = RenkoConsolidator(1000, lambda data: data.ask_size - bar.bid_size)
