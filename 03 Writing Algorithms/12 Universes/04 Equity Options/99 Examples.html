Example 1: 30-day Expiring Straddle

The following example selects an ATM call and an ATM put contract with at least 30 days from expiry to form a straddle.

public override void Initialize()
{
    var option = AddOption("SPY");
    option.SetFilter(optionFilterUniverse => optionFilterUniverse.IncludeWeeklys().Straddle(30));
}
def initialize(self):
    option = self.add_option("SPY")
    option.set_filter(lambda option_filter_universe: option_filter_universe.include_weeklys().straddle(30))

Example 2: 10-day Expiring Box Spread

A box spread strategy secures arbitrage profit from mispricing if there are any. The following example selects the 4 contracts to form a box spread with at least 10 days from expiry and $5 from ATM.

public override void Initialize()
{
    var option = AddOption("SPY");
    option.SetFilter(optionFilterUniverse => optionFilterUniverse.IncludeWeeklys().BoxSpread(10, 5));
}
def initialize(self):
    option = self.add_option("SPY")
    option.set_filter(lambda option_filter_universe: option_filter_universe.include_weeklys().box_spread(10, 5))

Example 3: 30-60-day Expiring Jelly Roll

A jelly roll strategy secures arbitration profit from interest rate difference if there exists any. The following example selects the 4 ATM contracts to form a jelly roll with 30 to 60 days from expiry.

public override void Initialize()
{
    var option = AddOption("SPY");
    option.SetFilter(optionFilterUniverse => optionFilterUniverse.IncludeWeeklys().JellyRoll(0, 30, 60));
}
def initialize(self):
    option = self.add_option("SPY")
    option.set_filter(lambda option_filter_universe: option_filter_universe.include_weeklys().jelly_roll(0, 30, 60))
