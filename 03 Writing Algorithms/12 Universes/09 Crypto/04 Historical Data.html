To get historical Cryptocurrency universe data, call the History method with the Universe object and the lookback period. The return type is a multi-index pandas.Series of CryptoUniverse list.

foreach (var universeDay in history)
{
    foreach (CryptoUniverse universeItem in universeDay)
    {
        Log($"{universeItem.Symbol} price at {universeItem.EndTime}: {universeItem.Close}");
    }
