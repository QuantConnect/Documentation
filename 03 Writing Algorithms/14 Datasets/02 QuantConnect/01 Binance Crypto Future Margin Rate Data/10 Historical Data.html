To get historical Binance Crypto Future Margin Rate data, call the History<MarginInterestRate>history method with MarginInterestRate data type and the Crypto Future Symbol. If there is no data in the period you request, the history result is empty.

# DataFrame
history_df = self.history(MarginInterestRate, self.btcbusd, 100, Resolution.DAILY)

# MarginInterestRate objects
history = self.history[MarginInterestRate](self.btcbusd, 100, Resolution.DAILY)
var history = History<MarginInterestRate>(_symbol, 100, Resolution.Daily);

For more information about historical data, see History Requests.
