LEAN supports the following Option price models.

Model OptionPriceModels American Style European Style
BinomialCoxRossRubinsteinbinomial_cox_ross_rubinstein
BlackScholesblack_scholes
ForwardTreeforward_tree

These models implement the IndicatorBasedOptionPriceModel, which uses the implied volatility and Greeks indicators, with different methodologies.

public override Initialize()
{
    var spyOption = AddOption("SPY");
    spyOption.PriceModel = OptionPriceModels.BinomialCoxRossRubinstein();
    var spxOption = AddIndexOption("SPX");
    spxOption.PriceModel = OptionPriceModels.BlackScholes();
}
def initialize(self):
    spy_option = self.add_option("SPY")
    spy_option.price_model = OptionPriceModels.binomial_cox_ross_rubinstein()
    spx_option = self.add_index_option("SPX")
    spx_option.price_model = OptionPriceModels.black_scholes()

If you set the price model of an Option to a model with an incompatible style, LEAN throws an exception.
