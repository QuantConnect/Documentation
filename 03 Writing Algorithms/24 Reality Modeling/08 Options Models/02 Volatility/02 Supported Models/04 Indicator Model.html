The IndicatorVolatilityModel sets the volatility of the security to the Current.Valuecurrent.value property of an indicator.

underlyingSecurity.VolatilityModel = new IndicatorVolatilityModel();
underlying_security.volatility_model = IndicatorVolatilityModel()

The following table describes the arguments the model accepts:

Argument Data Type Description Default Value
indicator IIndicator<T> The auto-updating indicator

To view the implementation of this model, see the LEAN GitHub repository.
