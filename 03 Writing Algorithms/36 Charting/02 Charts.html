Charts contain a collection of series, which display data on the chart. To add a chart to an algorithm, create a Chart object and then call the AddChartadd_chart method.

// Add a chart to an algorithm by creating a Chart object and then calling the AddChart method to visualize and monitor custom metrics or data over time.
var chart = new Chart("<chartName>");
AddChart(chart);
# Add a chart to an algorithm by creating a Chart object and then calling the add_chart method to visualize and monitor custom metrics or data over time.
chart = Chart("<chartName>")
self.add_chart(chart)

The Chart constructor expects a name argument. The following chart names are reserved:

  • Assets Sales Volume
  • Exposure
  • Portfolio Margin
