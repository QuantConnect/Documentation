Execution models can place orders for your strategy instead of placing them manually. To do so, PortfolioTargets are routed to the Executeexecute method of the Execution Model to place orders.

The following example uses a Framework Execution Model in a classic style algorithm:

public override void Initialize()
{
    // Execute the market order imediately to fill portfolio targets.
    SetExecution(new ImmediateExecutionModel());
}
def initialize(self):
    # Execute the market order imediately to fill portfolio targets.
    self.set_execution(ImmediateExecutionModel())
