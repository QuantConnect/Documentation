Follow these steps to import the mlfinlab library:

  1. Open a project.
  2. In the Project panel, click the LEAN Environment field and then click Hudson & Thames from the drop-down menu.
  3. Add your MlFinLab API key as a project parameter (for example, mlfinlab-api-key).

    4. Using a parameter keeps your API key safe for sharing project code.

  4. Open one of the code files in your project.
  5. At the top of the code file, add import ht_auth.
  6. In the Initialize method, add the following snippet: 
    ht_auth.SetToken("mlfinlab-api-key")
