Follow these steps to save tslearn models into the Object Store:

  1. Set the key name of the model to be stored in the Object Store.
    2. # Set the key name to save model.
model_key = "model.hdf5"
  2. Call the GetFilePathget_file_path method with the key.
    3. # Get the file path where model will be stored.
file_name = self.object_store.get_file_path(model_key)

    This method returns the file path where the model will be stored.

  3. Delete the current file to avoid a FileExistsError error when you save the model.
    4. import os
os.remove(file_name)
  4. Call the to_hdf5 method with the file path.
    5. # Serialize Python objects into a file.
self.model.to_hdf5(file_name)
