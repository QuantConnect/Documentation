Follow these steps to save keras models into the Object Store:

  1. Set the key name you want to store the model under in the Object Store. Add either a .keras extension for the native Keras format (recommended) or a .h5 extension.
    2. model_key = "model.keras"
  2. Call the GetFilePath method with the key.
    3. file_name = self.object_store.get_file_path(model_key)

    This method returns the file path where the model will be stored.

  3. Call the save method with the file path.
    4. self.model.save(file_name)
