Follow these steps to load the pre-trained Chronos-T5 model:

  1. Import the Chronos pipeline class.
    2. from chronos import ChronosPipeline
  2. Call the ChronosPipeline.from_pretrained method with the model path.

    3. In QuantConnect Cloud, the path to the tiny model is amazon / chronos-t5-tiny.

    self._pipeline = ChronosPipeline.from_pretrained(
    "amazon/chronos-t5-tiny",
    device_map="cuda" if torch.cuda.is_available() else "cpu",
    torch_dtype=torch.bfloat16,
)
  3. (Optional) Set the seed to enable reproducibility.
    4. from transformers import set_seed
set_seed(1, True)
