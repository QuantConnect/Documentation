Exchanges are located in different areas, so they can have different time zones. LEAN tracks the time zone of each exchange so that you receive the correct data at the right point in time in your algorithm. If you have multiple exchanges in your algorithm from different time zones, LEAN synchronizes them to your algorithm time. Furthermore, if you set up time period consolidators, LEAN consolidates the data based on the data time zone.

To get the time zone of the exchange that a Security trades on, use its Exchange.Hours.TimeZoneexchange.hours.time_zone property.

// Get the time zone of the exchange for a specific asset.
var timeZone = Securities[_symbol].Exchange.Hours.TimeZone;
# Get the time zone of the exchange for a specific asset.
time_zone = self.securities[self._symbol].exchange.hours.time_zone
