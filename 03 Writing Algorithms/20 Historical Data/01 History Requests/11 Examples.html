Example 1: Warm Up TradeBar Indicator

We can make use of historical traed bar to warm up a TradeBar indicator like AverageTrueRange.

private AveargeTrueRange _atr;
private Symbol _symbol = Symbol.Create("SPY", SecurityType.Equity, Market.USA);

public override void Initialize()
{
    _atr = new AverageTrueRange(14);
    var tradeBarHistory = History<TradeBar>(_symbol, 14, Resolution.Daily);
    foreach (TradeBar tradeBar in tradeBarHistory)
    {
        atr.Update(tradeBar);
    }
}
def initialize(self):
    self.atr = AverageTrueRange(14)
    self._symbol = Symbol.create("SPY", SecurityType.EQUITY, Market.USA)
    trade_bar_history = self.history[TradeBar](self._symbol, 14, Resolution.DAILY)
    for trade_bar in trade_bar_history:
        self.atr.update(trade_bar)

Example 3: Resampled History

You can also resample the historical data into different resolution, such as weekly in this example.

daily_history = self.history(["SPY", "QQQ"], 100, Resolution.DAILY)
weekly_history = daily_history.groupby(pd.Grouper(freq='W', level=-1))
