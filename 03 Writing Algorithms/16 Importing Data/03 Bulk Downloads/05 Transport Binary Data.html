Follow these steps to transport binary files:

  1. Add the following imports to your local program:
    2. import pickle
import base64
  2. Serialize your object.
    3. pickle_bytes = pickle.dumps(my_object)
base64_str = base64.b64encode(pickle_bytes).decode('ascii')
  3. Save the string representation of your object to one of the supported sources.
  4. Download the remote file into your project.
    5. base64_str = self.Download("<fileURL>")
  5. Restore the object.
    6. base64_bytes = base64_str.encode('ascii')
model = base64.b64decode(base64_bytes)
restored_model = pickle.loads(model)
