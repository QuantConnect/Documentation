To get the value of a runtime statistic in an algorithm, index the RuntimeStatisticsruntime_statistics member of the algorithm class with the statistic name. The values of the RuntimeStatisticsruntime_statistics dictionary are strings, so you may need to cast the result to a different data type.

// To monitor and manage runtime statistic metrics, index the runtime_statistic member with the statistic name to get the value of the runtime statistic while the algorithm is running.
var value = RuntimeStatistics[name];
