To get the StatisticsResults, use the Statisticsstatistics member of the algorithm class.

// Measure key portfolio perfomance metrics by using statistics member of the algorithm class to get StatisticsResults.
var sharpe = Statistics.TotalPerformance.PortfolioStatistics.SharpeRatio;
# Measure key portfolio perfomance metrics by using statistics member of the algorithm class to get StatisticsResults.
sharpe = self.statistics.total_performance.portfolio_statistics.sharpe_ratio

To get the value of a custom statistic while your algorithm runs, index the Summarysummary member with the statistic name. The values in the Summarysummary dictionary are strings, so you may need to cast the value that returns to a different data type.

// To monitor and manage custom statistic metrics, index the summary member with the statistic name to get the value of a custom statistic while the algorithm is running.
var value = Statistics.Summary["name"];
# To monitor and manage custom statistic metrics, index the summary member with the statistic name to get the value of a custom statistic while the algorithm is running.
value = self.statistics.summary["name"]
