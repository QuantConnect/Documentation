To create an Index subscription, in the Initializeinitialize method, call the AddIndexadd_index method. The AddIndexadd_index method returns an Index object, which contains a Symbol. Save a reference to the Symbol so you can use it in OnDataon_data to access the Index data in the Slice.

_symbol = AddIndex("VIX").Symbol;
self._symbol = self.add_index("VIX").symbol
