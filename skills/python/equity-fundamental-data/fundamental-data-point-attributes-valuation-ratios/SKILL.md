# Valuation Ratios attributes — QuantConnect / LEAN `Fundamental`

Reach these off a `Fundamental` snapshot `f` as `f.valuation_ratios.<attribute>` — for example `f.valuation_ratios.pe_ratio` or `f.valuation_ratios.ev_to_ebitda`. Get `f` from an `add_universe(...)` selection callback, from `self.securities["SPY"].fundamentals`, or from a history request.

Unlike the financial statements, valuation ratios are read **directly** — there is no period accessor. `f.valuation_ratios.pe_ratio` returns the number straight away; appending `.value` is an error. The `Type` column below shows what each returns (most are numbers). For how to navigate the rest of the tree, see the **fundamental-universes** skill.
