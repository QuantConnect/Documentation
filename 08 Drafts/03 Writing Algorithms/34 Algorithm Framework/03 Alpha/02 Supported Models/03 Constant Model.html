The ConstantAlphaModel always returns the same insight for each security.

AddAlpha(new ConstantAlphaModel(type, direction, period));
self.AddAlpha(ConstantAlphaModel(type, direction, period))

The following table describes the arguments the model accepts:

Argument Data Type Description Default Value
type InsightType The type of insight
direction InsightDirection The direction of the insight
period TimeSpantimedelta The period over which the insight will come to fruition
magnitude double?float/NoneType The predicted change in magnitude as a +/- percentage nullNone
confidence double?float/NoneType The confidence in the insight nullNone
weight double?float/NoneType The portfolio weight of the insights nullNone
