Unless you are working on an anonymous project, Local Platform automatically syncs your local project files with QuantConnect Cloud. Every time you save a file, Local Platform saves the changes in your local project and in the cloud version of the project.

To synchronize only the project you currently have open, or to disable synchronization entirely, adjust the Sync: Local And Cloud Projects extension setting. Disabling synchronization is best when local agents edit your project files and you want to avoid conflicts with the cloud versions of your projects.
