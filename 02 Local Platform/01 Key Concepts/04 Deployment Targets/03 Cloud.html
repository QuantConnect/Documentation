The Cloud deployment target is a collection of servers that the QuantConnect team manages. It's the same deployment target you use if you create projects, spin up research nodes, and deploy algorithms on the QuantConnect website. Follow these steps to set the deployment target of a project on Local Platform to Cloud:

  1. Create a project or open an exisiting one.
  2. In the Project panel, click the Deployment Target field and then click Cloud from the drop-down menu.

After you set the deployment target to Cloud, the Build and Backtest icons are orange. For more information about QuantConnect Cloud, including our quotas and security guidelines, see Cloud Platform.
