What is the gallery:
- We present examples of plots below.
- For a comprehensive list of supported plots for each plotting package, refer to their documnetation

Why do we have a gallery:
- To show what is possible when combining the plotting packages above with QC data


<h4>Candle
- What it is
    - A time series plot that shows the open, high, low, and close values for consolidated time periods
    - ex: 1 minute data stream builds 5 minutes candles
    - Up candles and down candles are different colors
- Why/when you would use it
    - When you want to see data consolidated into time periods (fixed or dynamic [renko])
    - It's a clean way to represent a lot of data with each figure (candle) added to the plot
         - Each candle represents 4 data points (OHLC)
- What it tells you
     - OHLC values for time periods
- Show an example and explain the data that's being used (dataset name)

<h4>Line
- What it is
       - A line connecting data points in an ordered fashion (time series)
- Why/when you would use it
       - It's a simple (common) way to plot a time series
- What it tells you
       - The value of some dataset across time/samples
- Show an example and explain the data that's being used (dataset name)

<h4>Scatter
- What it is
        - A plot to show the relationship between two variables
        - Can be 2-d or 3-d
- Why/when you would use it
        - To see what happens to variable y when variable x changes
- What it tells you
        - The relationship between two variables
        - Correlation
- Show an example and explain the data that's being used (dataset name)

<h4>Histogram
- What it is
        - A representation of how often numerical data occurs within thresholds (buckets)
- Why/when you would use it
        - To see the distribution of the data points
        - To see if one bucket occurs more frequently than another bucket
- What it tells you
        - How many data points occurred in each bucket
- Show an example and explain the data that's being used (dataset name)

<h4>Bar
- What it is
          - "A bar chart or bar graph is a chart or graph that presents categorical data with rectangular bars with heights or lengths proportional to the values that they represent. The bars can be plotted vertically or horizontally." // wiki definition
- Why/when you would use it
          - When you want to see how often each category in the data occurs (number of up days vs down days)
          - When you want to see the proportion of each category in the data (% up days vs % down days)
- What it tells you
          - How often different categories in the data occur relative to eachother
- Show an example and explain the data that's being used (dataset name)

<h4>Box
- What it is
       - "A method for graphically depicting groups of numerical data through their quartiles." // wiki definition
- Why/when you would use it
       - When you want to see the variability in the dataset. Smaller boxes => Less variability
- What it tells you
        - Minimum, Maximum, median, 25th percentile, and 75th percentile of a dataset
- Show an example and explain the data that's being used (dataset name)

<h4>Violin
- What it is
       - "Violin plots are similar to box plots, except that they also show the probability density of the data at different values, usually smoothed by a kernel density estimator." // wiki definition
- Why/when you would use it
       - When you want to see the variability of the dataset and the frequency of each value within the range
- What it tells you
       - Minimum, Maximum, median, 25th percentile, and 75th percentile of a dataset
       - The frequency of each value between the min and max
- Show an example and explain the data that's being used (dataset name)

<h4>Heatmap
- What it is
        - A 2d grid to that shows the magnitude of each entry in the grid as a color on a scale
- Why/when you would use it
        - To aid in understanding the correlation between variables
        - An alternative to looking at a grid of numbers, look at the colors to see patterns/relationships/anomalies
- What it tells you
        - The magnitude of each value in the grid in color format
- Show an example and explain the data that's being used (dataset name)

<h4>3D surface
- What it is
        - A 3 dimensional plot
- Why/when you would use it
        - To visualize the relationship between three variables
        - Ex: To see how the Sharpe ratio (vertical axis) changes when you adjust the value of the parameters on the x and y axes
- What it tells you
        - How the value of the vertical axis changes when you adjust either the x or y axes
- Show an example and explain the data that's being used (dataset name)

<h4> Pie Chart
- What it is
       - A 2D chart that shows the share of each component of a group
- Why/when you would use it
       - To visualize the importance of a component
       - Ex: To see which security is the most important in the portfolio
