Follow these steps to subscribe to an alternative dataset from the Dataset Market:

  1. Load the required assembly files and data types.
    2. #load "../Initialize.csx"
#load "../QuantConnect.csx"

using QuantConnect;
using QuantConnect.Data;
using QuantConnect.Algorithm;
using QuantConnect.Research;
using QuantConnect.DataSource;
  2. Load the dynamic link library (DLL) of the dataset.

    3. To load the DLL of any dataset, type:

    #r "../QuantConnect.DataSource.<nameOfAlternativeDatasetClass>.dll"

    For example, to load the DLL of the CBOE dataset, type:

    #r "../QuantConnect.DataSource.CBOE.dll"
  3. Instantiate a QuantBook.
    4. var qb = new QuantBook();
    qb = QuantBook()
  4. Call the AddData method with the dataset class, a ticker, and a resolution.
    5. var cboe = qb.AddData<CBOE>("VIX", Resolution.Daily);
    cboe = qb.AddData(CBOE, "VIX", Resolution.Daily)

    To view the arguments that the AddData method accepts for the dataset you're using, see the dataset listing.

    If you do not pass a resolution argument, the default resolution of the dataset is used by default. To view the supported resolutions and the default resolution of the dataset you're using, see the dataset listing.

  5. Save a reference to the alternative data Symbol.
    6. var symbol = cboe.Symbol;
    symbol = cboe.Symbol
