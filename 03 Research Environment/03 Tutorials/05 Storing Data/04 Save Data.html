You need storage create permissions to save data to the ObjectStore.

SaveBytes

Call the SaveBytes method could store the byte list object.

objectStore.SaveBytes("saveBytes", bytesSample)
object_store.SaveBytes("save_bytes", bytes_sample)

The following method is the helper method of SaveBytes that is internally called by it:

Save

Call the Save or SaveString method could store data as a string object.

objectStore.Save("saveString", stringSample);
object_store.Save("save_string", string_sample)

SaveJson

Call the SaveJson<T> method to store data as a JSON object. The helper method will help serialize the data into JSON format automatically.

objectStore<Dictionary<string, int>>.SaveJson("saveJson", sample);

SaveXml

Call the SaveXml<T> method to store data the XML object. 

objectStore.SaveXml("saveXml", xmlSample);
