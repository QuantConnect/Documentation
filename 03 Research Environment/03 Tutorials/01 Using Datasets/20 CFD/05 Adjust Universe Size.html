You need to subscribe to all of the CDFs for which you want historical data. You can request historical data for a subset of the CFDs or all of the CFDs.

Subset of CFDs

Create subscriptions for multiple CFDs and then call the History method with a list of Symbols to request historical data for those specific securities.

var qb = new QuantBook();
var b02y = qb.AddCfd("USB02YUSD").Symbol;
var b05y = qb.AddCfd("USB05YUSD").Symbol;
var b10y = qb.AddCfd("USB10YUSD").Symbol;

var startTime = new DateTime(2021, 1, 1);
var endTime = new DateTime(2021, 2, 1);
var definedPeriodHistory = qb.History(new []{b02y, b05y, b10y}, startTime, endTime, Resolution.Daily);
var timespanHistory = qb.History(new []{b02y, b05y, b10y}, TimeSpan.FromDays(10), Resolution.Daily);
var barCountHistory = qb.History(new []{b02y, b05y, b10y}, 10, Resolution.Daily);
qb = QuantBook()
b02y = qb.AddCfd("USB02YUSD").Symbol
b05y = qb.AddCfd("USB05YUSD").Symbol
b10y = qb.AddCfd("USB10YUSD").Symbol

start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
defined_period_history = qb.History([b02y, b05y, b10y], start_time, end_time, Resolution.Daily)
timespan_history = qb.History([b02y, b05y, b10y], timedelta(days=10), Resolution.Daily)
bar_count_history = qb.History([b02y, b05y, b10y], 10, Resolution.Daily)

All CFDs

Call the History method with no Symbols to request historical data for all of the CFDs.

Call the History method with qb.Securities.Keys, time argument(s), and resolution to request historical data for all of the CFDs.

var startTime = new DateTime(2021, 1, 1);
var endTime = new DateTime(2021, 2, 1);
var definedPeriodHistory = qb.History(startTime, endTime, Resolution.Daily);
var timespanHistory = qb.History(TimeSpan.FromDays(10), Resolution.Daily);
var barCountHistory = qb.History(10, Resolution.Daily);
start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
defined_period_history = qb.History(qb.Securities.Keys, start_time, end_time, Resolution.Daily)
timespan_history = qb.History(qb.Securities.Keys, timedelta(days=10), Resolution.Daily)
bar_count_history = qb.History(qb.Securities.Keys, 10, Resolution.Daily)

The History method returns an IEnumerable<Slice> for multiple-security requests.
